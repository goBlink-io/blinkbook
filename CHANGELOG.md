# Changelog

All notable changes to BlinkBook are documented here.

## [0.5.0] — 2026-03-02

### Added
- Multi-step onboarding wizard with first page creation
- Welcome email on signup
- Custom 404 and error pages
- Loading skeleton components across dashboard
- Empty state components for spaces and pages
- Global keyboard shortcuts (Cmd+K, Cmd+N, Cmd+S, Cmd+P, ?)
- Keyboard shortcut help modal
- SEO noindex for dashboard routes
- Vercel deployment configuration
- Environment variable documentation

### Improved
- Next.js image optimization config
- Production console log removal
- Bundle analysis script

## [0.4.0] — 2026-02-15

### Added
- Stripe billing integration (Free, Pro, Team plans)
- goBlink crypto payment support
- Team management with invitations
- Role-based access control (owner, admin, editor, viewer)
- Plan enforcement and usage limits
- Billing dashboard with subscription management

## [0.3.0] — 2026-02-01

### Added
- Published site renderer with custom themes
- Full-text search with FlexSearch
- Custom domain support
- Page analytics and pageview tracking
- Template gallery with starter docs
- Pricing page

## [0.2.0] — 2026-01-15

### Added
- Tiptap block editor with slash commands
- Floating toolbar for text formatting
- Page tree with drag-and-drop reordering
- Editor sidebar with collapsible navigation
- Auto-save with debounce
- Publish/unpublish toggle per page
- Code block with syntax highlighting (20+ languages)
- Callout blocks (info, warning, danger)
- Image, table, and horizontal rule blocks

## [0.1.0] — 2026-01-01

### Added
- Supabase authentication (login, signup)
- Dashboard with space management
- Space creation wizard (name, slug, theme)
- 6 theme presets (Midnight, Ocean, Forest, Sunset, Lavender, Arctic)
- Dashboard sidebar navigation
- Account and billing page shells
