## Overview

This page demonstrates BlinkBook's API documentation features, including the interactive API playground.

## List Items

Retrieve a list of items from the API.

<ApiPlayground
  method="GET"
  endpoint="/api/items"
  baseUrl="https://jsonplaceholder.typicode.com"
  headers={{ "Accept": "application/json" }}
/>

## Create Item

Create a new item via the API.

<ApiPlayground
  method="POST"
  endpoint="/api/items"
  baseUrl="https://jsonplaceholder.typicode.com"
  headers={{ "Content-Type": "application/json" }}
  bodySchema={[
    { name: "title", type: "string", required: true, placeholder: "My Item", description: "The item title" },
    { name: "body", type: "string", placeholder: "Item description", description: "The item body text" },
    { name: "userId", type: "number", required: true, placeholder: "1", description: "The user ID" },
  ]}
/>

## Response Codes

| Status | Description |
|--------|------------|
| 200 | Success |
| 201 | Created |
| 400 | Bad Request |
| 401 | Unauthorized |
| 404 | Not Found |
| 500 | Server Error |
