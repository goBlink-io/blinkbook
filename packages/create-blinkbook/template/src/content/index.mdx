## Welcome to Your Docs

This is your new documentation site built with **BlinkBook** — a fast, customizable, open-source documentation framework.

## Getting Started

Check out the [Getting Started](/docs/getting-started) guide to learn how to customize your docs.

## Features

- **MDX Content** — Write docs in Markdown with JSX components
- **Full-Text Search** — Client-side search with zero external services
- **Theme Presets** — 6 built-in themes or bring your own colors
- **API Playground** — Interactive API testing right in your docs
- **Code Blocks** — Syntax highlighting with copy buttons
- **Responsive** — Mobile-optimized from day one
